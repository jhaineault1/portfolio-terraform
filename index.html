How This Site Was Built

I built this site using a "Serverless" CI/CD pipeline in AWS. The domain is managed by AWS's Route 53 while CloudFront hosts the site. The portfolio files are stored in a S3 bucket which were mainly written in ReactJS and BABEL. AWS's CodeBuild grabs the code from my GitHub and automatically pushes it into production using AWS's CodePipeline combined with a Lambda function. Once the code reaches the S3 bucket a SNS(Simple Notification Service) sends an email stating whether the code was deployed successfully or not.
